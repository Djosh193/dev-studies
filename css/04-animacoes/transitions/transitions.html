CSS Transitions (Visualizador)

Exemplos de transition em hover, focus e active: duração, delay, timing-function, múltiplas propriedades, all, transform/opacity, menu, card e imagem.

Dica: passe o mouse, use TAB para focar inputs/botões e clique/segure para ver :active.

1) Básico (hover)

Cor (background)

transition: background-color 250ms ease;

Opacity

transition: opacity 300ms ease;

FADE

2) Transform (melhor performance)

Translate (levantar)

transition: transform 220ms ease;

Scale

transition: transform 220ms ease;

SCALE

3) Timing functions (curvas)

Passe o mouse em cada faixa e compare o “jeito” do movimento.

linear
ease
ease-in
ease-out
ease-in-out
cubic-bezier

4) Delay

Com delay

transition: transform 250ms ease 250ms;

Sem delay

transition: transform 250ms ease;

5) Múltiplas propriedades

Card (transform + shadow + border)

transition: transform, box-shadow, border-color;

Card com hover

Sutil, mas passa sensação de “clicável”.

“transition: all” (use com cuidado)

transition: all 280ms ease;

ALL

6) Menu (color + underline)

Início Projetos Contato

Transição em color e um underline animado com pseudo-elemento.

7) Imagem (zoom no hover)

Zoom suave com transform: scale() (boa performance).

8) Focus e Active

Input com focus

Use TAB e veja a borda/sombra mudarem suavemente.

Botão com active (clique e segure)

Active geralmente usa transição curta e “resposta rápida”.

9) Boas práticas

  • Prefira animar transform e opacity.
  • Evite animar sempre width, height, top/left.
  • Interações de UI costumam ficar boas com 180–350ms.
  • Respeite prefers-reduced-motion para acessibilidade.
