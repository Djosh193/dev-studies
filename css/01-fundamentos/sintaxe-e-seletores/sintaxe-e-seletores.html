Sintaxe e Seletores no CSS

Exemplos visuais de: tag, classe, id, agrupamento, combinadores, atributos, pseudo-classes e pseudo-elementos.

1) Seletores básicos

Seletor de TAG

O CSS estiliza todas as tags p e h3.

Sou um H3 (tag)

Eu sou um parágrafo (tag p).

Outro parágrafo (tag p).

Seletor de CLASSE

O CSS estiliza qualquer elemento com .card-alvo.

Sou uma div com .card-alvo

Sou um p com .card-alvo

Seletor de ID

O CSS estiliza o elemento com #destaque (único).

Eu sou o #destaque

Eu não sou o destaque.

Agrupamento ( , )

O CSS aplica estilo em h4, kbd e mark ao mesmo tempo.

Sou um H4

Atalho: Ctrl + S

Texto destacado com mark

2) Combinadores de seletores

Descendente (A B)

Exemplo: .area-desc p estiliza qualquer p dentro de .area-desc.

Eu estou dentro de .area-desc

Eu também (descendente, mesmo estando “mais fundo”).

Filho direto (A > B)

Exemplo: .area-filho > p estiliza só filhos diretos.

Sou filho direto ✅

Eu NÃO sou filho direto (sou neto) ❌

Irmão adjacente (A + B)

Exemplo: .titulo + p estiliza o primeiro p imediatamente após .titulo.

Título

Eu sou o 1º p logo depois do título ✅

Eu sou o 2º p (não pego) ❌

Irmãos gerais (A ~ B)

Exemplo: .marco ~ p estiliza todos os p após .marco.

Antes do marco (não pega).

Marco

Depois do marco ✅

Depois do marco ✅

Depois do marco ✅

3) Seletores de atributo

[atributo="valor"]

Exemplo: input[type="text"] e input[type="password"].

Parciais (^=, $=, *=)

Exemplo: a[href^="https"], a[href$=".pdf"], a[href*="github"].

Link HTTPS Link PDF Link GitHub Link HTTP (não pega ^="https")

4) Pseudo-classes

:hover, :focus, :active, :visited

Passe o mouse, clique e use TAB para focar.

Link de teste (visited muda depois)

:first-child, :last-child, :nth-child()

Uma lista onde o CSS aplica estilos por posição.

  • Primeiro item
  • Segundo item
  • Terceiro item
  • Último item

5) Pseudo-elementos

::first-letter e ::first-line

O CSS altera a primeira letra e a primeira linha.

Este parágrafo tem pseudo-elementos. A primeira letra fica maior e a primeira linha fica mais destacada. Redimensione a janela para mudar o que é “primeira linha”.

::before e ::after

O CSS adiciona conteúdo antes e depois sem alterar o HTML.

Novo

O “brilhinho” e a setinha são ::before/::after.

6) Especificidade (prioridade)

TAG vs CLASSE vs ID

No CSS: a tag pinta de azul, a classe pinta de verde, o ID pinta de vermelho. O texto final fica vermelho por causa da prioridade.

Eu tenho tag p + classe .texto + ID #principal

Combinação (mais específico)

.card-especifica p ganha de p.

Este p está dentro de .card-especifica
