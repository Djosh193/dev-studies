Box Model na prática

Comparação entre content-box (padrão) e border-box (recomendado).

Mesma largura declarada: width: 320px

Ambos têm padding: 24px e border: 8px, mas o resultado final muda.

❌ Padrão (content-box)

O width vale só para o conteúdo. O padding e a borda somam no tamanho final.

320px (referência)
content-box width: 320px
padding: 24px
border: 8px

Tamanho total = 320 + (24*2) + (8*2) = 384px

✅ Recomendado (border-box)

O width inclui conteúdo + padding + borda. O elemento respeita exatamente os 320px.

320px (referência)
border-box width: 320px
padding: 24px
border: 8px

Tamanho total = 320px
