2) Unidades

Unidades absolutas (px)

A caixa abaixo tem 300px × 200px.

width: 300px;
height: 200px;
font-size: 16px;

em vs rem (fonte)

em depende do tamanho de fonte do elemento pai.
rem depende do tamanho de fonte do html.

Pai: 20px

Texto 2em (fica 40px)

container-em { font-size: 20px }
p { font-size: 2em }

Root (html): 16px

Texto 2rem (fica 32px)

p { font-size: 2rem }

% (relativo ao pai)

A caixa interna tem 50% da largura e altura do pai.

Pai (fixo): 360px × 240px
Filho: 50% × 50%

Viewport units (vw / vh / vmin / vmax)

Redimensione a janela do navegador e veja as caixas mudarem.

width: 50vw

50vw = 50% da largura da tela

height: 20vh

20vh = 20% da altura da tela

width: 30vmin; height: 30vmin

vmin = menor dimensão (largura ou altura)

width: 30vmax; height: 30vmax

vmax = maior dimensão (largura ou altura)

Layout responsivo com % + rem

Eu tenho: width: 80%, max-width: 1200px, padding em rem e margin auto.

Isso é uma base comum de container em sites.
