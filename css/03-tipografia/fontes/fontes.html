Fontes no CSS (Visualizador)

Comparação prática de font-family, stacks com fallback, famílias genéricas, font-size, font-weight, font-style, font-variant, shorthand font, system fonts e monospace.

Dica: selecione o texto e compare legibilidade. Abra o DevTools (F12) e inspecione cada bloco.

1) font-family + fallback

Stack com fallback (recomendado)

font-family: "Inter", "Roboto", Arial, sans-serif;

Título de exemplo

Lorem ipsum dolor sit amet, consectetur adipiscing elit. 0123456789 — símbolos: @ # $ % & * ( ) .

Família genérica

font-family: serif | sans-serif | monospace

Serif: texto com serifa para comparação.

Sans-serif: comum em interfaces.

Monospace: ideal para código.
