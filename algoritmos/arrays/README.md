# Arrays em JavaScript

Arrays são estruturas de dados que armazenam múltiplos valores em uma única variável.

## Conceitos abordados

- Criação de arrays
- Acesso por índice
- Iteração
- Métodos nativos
- Busca
- Ordenação
- Problemas clássicos de algoritmos

## Complexidade

Acesso por índice: O(1)
Busca: O(n)
Inserção/remoção: depende da posição
